Telecast is a self-hosted static site generator that mirrors your Telegram channel into a fast, searchable, and beautifully designed microblog website. Built with Next.js, React, shadcn/ui, and Tailwind CSS, it outputs pure static HTML — meaning you can host it anywhere: Cloudflare Pages, Vercel, Netlify, GitHub Pages, or your own server.

## What Telecast does

Telecast connects to your public Telegram channel and, at build time, fetches all posts, downloads every image and video to `public/media/`, and compiles a fully static website with no runtime server required. Every page is pre-rendered HTML — fast, portable, and easy to self-host.

## Get started

<Steps>
  <Step title="Fork and clone">
    Fork the [andatoshiki/telecast](https://github.com/andatoshiki/telecast) repository and clone it locally.
  </Step>
  <Step title="Edit the configuration">
    Open `src/lib/constant.ts` and set your `channel` username and `siteUrl`. That's the minimum required configuration.
  </Step>
  <Step title="Run the sync">
    Run `pnpm sync` to fetch your channel content, mirror media files, and build the search index.
  </Step>
  <Step title="Build and deploy">
    Run `pnpm build` and deploy the output to your preferred static host.
  </Step>
</Steps>

<Tip>
  The full `pnpm build` script automatically runs `pnpm sync --og-image --favicon` before `next build`, so a single command generates all artifacts and produces a ready-to-deploy site.
</Tip>
